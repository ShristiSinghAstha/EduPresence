The Attendance Management System is a web-based application designed to digitally manage and monitor student attendance. The system replaces traditional manual attendance methods with a secure and efficient platform that allows administrators, faculty members, and students to manage attendance records easily.

By using role-based authentication and secure APIs, the system ensures transparency, improves academic monitoring, and reduces manual workload.

🚀 Features

🔐 Role-based login system (Admin, Faculty, Student)

📝 Digital attendance marking

📊 Attendance percentage calculation

📈 Attendance reports and analytics

👨‍🏫 Faculty attendance management

👨‍🎓 Student attendance tracking

🔎 Easy data retrieval and monitoring

🔒 Secure authentication and API access

🧑‍💻 User Roles
1️⃣ Admin

Manage users (students and faculty)

Create and manage classes

Assign subjects to faculty

View attendance reports

Monitor system activities

2️⃣ Faculty

Mark attendance for students

Update attendance records

View class-wise attendance

Generate attendance reports

3️⃣ Student

View personal attendance

Check attendance percentage

View subject-wise attendance
